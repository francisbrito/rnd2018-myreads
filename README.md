# My Reads
> A single page application for keeping track of the books I've read.

![Screenshot](./assets/scrot.png)

Track the books you're dying to read, easily.

## Installation
```sh
$ npm install
```

Or, if using `yarn`:
```sh
$ yarn install
```

## Usage
```sh
$ npm start
```

If using `yarn`:
```sh
$ yarn install
```
